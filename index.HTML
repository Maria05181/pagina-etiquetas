¿Qué son las etiquetas?

Las etiquetas ayudan a organizar y filtrar la información de manera eficiente. Son claves para mejorar la experiencia de navegación y encontrar contenido relevante rápidamente.

FIGURE

Contenedor para una imagen o contenido multimedia, junto con su pie de foto.

<figure>...</figure>
<>

FIGCAPTION

Define un pie de foto para una imagen o contenido dentro de la etiqueta.

<figcaption>...</figcaption>

MARK

Resalta texto importante o que se desea destacar.

<mark>...</mark>

PROGRESS

Define una barra de progreso.

<progress value="50" max="100"></progress>

METER

Representa un valor dentro de un rango determinado.

<meter value="0.6"></meter>

DETAILS

Define un bloque de contenido que el usuario puede mostrar u ocultar.

<details>...</details>

SUMMARY

Define un encabezado para el contenido dentro de la etiqueta.

<summary>...</summary>

CANVAS

Define un área de dibujo en la que se puede dibujar mediante JavaScript.

<canvas id="myCanvas">...</canvas>

DATALIST

Define un conjunto de opciones predefinidas para un campo de entrada.

<datalist id="browsers">...</datalist>

INPUT TYPE

Define un control deslizante para elegir un valor dentro de un rango.

<input type="range" min="0" max="100">
